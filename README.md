# Калькулятор сложного процента (Flask)

## Описание
Web-приложение для расчета сложного процента с различными вариантами частоты начисления процентов.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python main.py
```

3. Откройте браузер и перейдите по адресу:
```
http://127.0.0.1:5000
```

## Функционал

- Расчет итоговой суммы по формуле сложного процента: **A = P × (1 + r/n)^(n×t)**
- Выбор частоты начисления процентов:
  - Раз в год
  - Раз в полгода
  - Ежеквартально
  - Ежемесячно
  - Ежедневно
- Валидация введенных данных
- Красивый и адаптивный интерфейс

## Параметры расчета

- **P** (Principal) — начальная сумма вклада
- **r** (Rate) — годовая процентная ставка (в процентах)
- **n** (Frequency) — частота начисления процентов в год
- **t** (Time) — срок вклада в годах

## Пример использования

Входные данные:
- Начальная сумма: 100 000 руб.
- Годовая ставка: 10%
- Срок: 5 лет
- Частота: ежемесячно (12 раз в год)

Результат:
- Итоговая сумма: 164 530,89 руб.
- Начисленные проценты: 64 530,89 руб.
